  "The Ultimate Intelligent Golf Management Platform"

📖 ภาพรวมระบบ (System Overview)
    Green Valley System เป็นระบบที่ออกแบบตามหลัก Object-Oriented Programming (OOP) เพื่อรองรับการดำเนินงานของสนามกอล์ฟยุคใหม่ โดยหัวใจหลักของระบบประกอบด้วย:
    Booking Engine: ระบบจอง Tee-off ที่มีกฎการจองล่วงหน้าตาม Tier ของสมาชิก (Platinum/Gold/Silver).
    Tournament Management: จัดการแข่งขันกอล์ฟแบบเต็มรูปแบบ ตั้งแต่การสมัคร, การจัดกลุ่ม (Pairing), บันทึกคะแนนแบบ Real-time ไปจนถึงการตัดเกรด Handicap.
    Resource Orchestration: ระบบจัดการแคดดี้และรถกอล์ฟ โดยมีการตรวจสอบความว่าง (Availability) แบบ 5-Hour Rule เพื่อป้องกันการจองซ้อน.
    Smart Penalty (Strike System): ระบบลงโทษสมาชิกที่ผิดกฎ (เช่น No-show) โดยอัตโนมัติ

✨ ฟีเจอร์เด่น (Key Features)
    ⛳ สำหรับนักกอล์ฟ (Golfer Experience)
        Tier-based Benefits: สมาชิกจะได้รับส่วนลดและสิทธิ์การจองล่วงหน้าที่แตกต่างกันตามสถานะ (เช่น Platinum จองล่วงหน้าได้ 30 วัน).
        Rain Check System: หากเกิดเหตุสุดวิสัย (ฝนตก) ระบบสามารถออกคูปองส่วนลดเพื่อใช้ในครั้งถัดไปได้.
        Handicap Tracking: คำนวณแต้มต่ออัตโนมัติจากประวัติการเล่นย้อนหลัง (WHS Standard).
    🛠 สำหรับระบบ (System)
        Real-time Leaderboard: ตารางคะแนนสดระหว่างการแข่งขัน.
        Inventory Management: ระบบขายสินค้าและอาหาร (F&B) ที่ตัดสต็อกทันทีเมื่อมีการสั่งซื้อผ่านการจอง.
        Automated Notifications: ระบบแจ้งเตือนสถานะการจองและการชำระเงินหาผู้ใช้โดยตรง.
